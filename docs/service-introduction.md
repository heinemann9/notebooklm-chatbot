# NotebookLM 챗봇 서비스 소개

> 문서 기반 AI 채팅 서비스 — 업로드한 자료를 바탕으로 질문에 답변하는 AI 챗봇

---

## 이 서비스는 무엇인가요?

NotebookLM 챗봇은 **우리가 가진 문서를 AI에게 학습시킨 뒤, 그 내용을 바탕으로 질문에 답변**해주는 서비스입니다.

일반적인 ChatGPT와 달리, **우리가 제공한 자료만을 근거로 답변**하기 때문에 정확하고 신뢰할 수 있는 응답을 받을 수 있습니다.

### 활용 예시

- 사내 매뉴얼을 업로드하고, 직원들이 궁금한 점을 채팅으로 질문
- 제품 가이드를 등록하고, 고객이 사용법을 직접 검색하는 대신 대화형으로 안내받기
- 교육 자료를 올려두고, 학습자가 자유롭게 질문하며 학습

---

## 역할 소개

이 서비스에는 **관리자**와 **고객**, 두 가지 역할이 있습니다.

| 구분 | 관리자 | 고객 |
|------|--------|------|
| 로그인 | 필요 (비밀번호) | 불필요 |
| 주요 활동 | 노트북 생성, 자료 등록, 채팅 테스트 | AI에게 질문하고 답변 받기 |
| 접속 주소 | 서비스 메인 페이지 | 별도 채팅 페이지 (링크 공유받음) |

---

## 관리자 이용 가이드

관리자는 AI가 참고할 자료를 등록하고, 고객에게 채팅 링크를 공유하는 역할입니다.

### 1단계: 로그인

1. 서비스 주소에 접속하면 **로그인 페이지**가 나타납니다
2. 전달받은 **관리자 비밀번호**를 입력합니다
3. 로그인에 성공하면 **대시보드(노트북 목록)** 화면으로 이동합니다

### 2단계: 노트북 만들기

> "노트북"은 하나의 주제를 담는 공간입니다. 예: "신입사원 안내", "제품 FAQ" 등

1. 대시보드에서 **노트북 만들기** 버튼을 클릭합니다
2. 노트북 이름을 입력합니다 (예: `고객 지원 FAQ`)
3. 생성된 노트북이 목록에 나타납니다

### 3단계: 자료(소스) 등록하기

> AI가 답변할 때 참고하는 자료입니다. 많이 등록할수록 더 풍부한 답변이 가능합니다.

1. 대시보드에서 만든 **노트북을 클릭**하여 채팅 페이지로 이동합니다
2. 오른쪽 **소스 관리 패널**에서 자료를 추가합니다:
   - **URL 추가**: 웹 페이지 주소를 입력하면 해당 페이지의 내용을 자동으로 가져옵니다
   - **파일 업로드**: PDF, 문서 파일 등을 직접 올릴 수 있습니다
3. 업로드가 진행되면 **진행 상태**가 표시되며, 완료되면 소스 목록에 추가됩니다
4. 더 이상 필요 없는 소스는 **삭제 버튼**으로 제거할 수 있습니다

### 4단계: 채팅 테스트

1. 같은 채팅 페이지의 **대화창**에서 질문을 입력해봅니다
2. AI가 등록된 자료를 바탕으로 답변합니다
3. 답변이 만족스러우면 고객에게 링크를 공유할 준비가 된 것입니다

### 5단계: 고객에게 채팅 링크 공유

고객용 채팅 페이지 주소를 공유합니다:

```
[서비스 주소]/w/chat
```

이 링크는 로그인 없이 누구나 접속하여 바로 채팅을 시작할 수 있습니다.

## 고객 이용 가이드

고객은 관리자가 공유한 링크를 통해 AI 채팅을 이용합니다.

### 사용 방법

1. 관리자로부터 전달받은 **채팅 링크**에 접속합니다
2. 화면에 **주제(노트북) 목록**이 나타나면, 질문하고 싶은 주제를 선택합니다
   - 주제가 하나만 설정된 경우, 선택 없이 바로 채팅이 시작됩니다
3. **환영 메시지**가 표시되면 채팅이 준비된 것입니다
4. 하단 입력창에 **궁금한 내용을 자유롭게 질문**합니다
5. AI가 등록된 자료를 바탕으로 답변합니다

### 알아두면 좋은 점

- **로그인이 필요 없습니다** — 링크만 있으면 바로 사용 가능합니다
- **이전 질문을 기억합니다** — 같은 세션 내에서는 앞선 대화 내용을 이어갈 수 있습니다
- **페이지를 새로고침하면 대화가 초기화됩니다** — 중요한 답변은 미리 복사해두세요
- **등록된 자료 범위 내에서 답변합니다** — 자료에 없는 내용은 답변하지 못할 수 있습니다

## 자주 묻는 질문 (FAQ)

### 일반

**Q. 일반 AI 챗봇(ChatGPT 등)과 뭐가 다른가요?**
> 일반 AI는 인터넷 전체에서 학습한 지식으로 답변합니다. 이 서비스는 **우리가 직접 등록한 자료만** 참고하여 답변하므로, 우리 조직/제품에 맞는 정확한 정보를 제공합니다.

**Q. 답변이 틀리면 어떻게 하나요?**
> AI는 등록된 자료를 바탕으로 최선의 답변을 생성하지만, 완벽하지 않을 수 있습니다. 답변의 정확도를 높이려면 관리자가 더 상세하고 정확한 자료를 등록하면 됩니다.

### 관리자

**Q. 어떤 자료를 올릴 수 있나요?**
> 웹 페이지 URL과 파일(PDF 등)을 업로드할 수 있습니다. 텍스트 형태의 자료라면 대부분 등록 가능합니다.

**Q. 자료를 수정하고 싶으면 어떻게 하나요?**
> 기존 소스를 삭제하고, 수정된 자료를 다시 업로드하면 됩니다.

**Q. 여러 주제를 운영하고 싶어요.**
> 주제별로 노트북을 따로 만들면 됩니다. 각 노트북에 해당 주제의 자료만 등록하면, 고객이 채팅 시 주제를 선택할 수 있습니다.

### 고객

**Q. 대화 내용이 저장되나요?**
> 브라우저를 닫거나 새로고침하면 대화 내용이 사라집니다. 필요한 답변은 직접 복사해두시기 바랍니다.

**Q. 모바일에서도 사용할 수 있나요?**
> 네, 모바일 브라우저에서 링크에 접속하면 동일하게 이용할 수 있습니다.

## 제한사항 및 유의사항

### 서비스 안정성

- **비공식 API 기반**: 이 서비스는 Google NotebookLM의 비공식 연동 방식을 사용합니다. Google 측 변경에 의해 **예고 없이 일시적으로 동작하지 않을 수 있습니다.**
- **인증 만료**: 내부적으로 Google 인증을 사용하며, 일정 시간이 지나면 인증이 만료됩니다. 이 경우 관리자가 **재인증**을 해야 서비스가 정상 동작합니다.
- **접속 주소 변경 가능**: 외부 접속용 주소(ngrok)는 서버 재시작 시 변경될 수 있으며, 변경 시 별도 안내됩니다.

### 대화 관련

- **대화 내용 보존 안 됨**: 고객 채팅은 브라우저 세션 내에서만 유지됩니다. **페이지를 새로고침하거나 브라우저를 닫으면 대화가 모두 사라집니다.** 중요한 답변은 반드시 복사해두세요.
- **등록된 자료 범위 내 답변**: AI는 관리자가 등록한 자료만 참고합니다. 자료에 포함되지 않은 내용에 대해서는 답변하지 못하거나 부정확할 수 있습니다.
- **답변 정확도**: AI가 생성하는 답변은 참고용이며, 100% 정확하다고 보장할 수 없습니다. 중요한 의사결정에는 반드시 원본 자료를 직접 확인하세요.

### 자료 등록 관련

- **동시 업로드 제한**: 여러 파일을 한꺼번에 업로드할 경우, 처리 시간이 길어지거나 일부가 실패할 수 있습니다. 한 번에 하나씩 등록하는 것을 권장합니다.
- **자료 수정 불가**: 이미 등록된 자료는 직접 수정할 수 없습니다. 내용을 변경하려면 기존 소스를 삭제한 뒤 다시 업로드해야 합니다.
- **지원 형식**: 웹 페이지(URL)와 문서 파일(PDF 등)을 지원합니다. 이미지만으로 구성된 파일이나 스캔 문서는 내용을 인식하지 못할 수 있습니다.

## 접속 안내

서비스에 접속하기 위한 주소는 운영 담당자에게 문의하세요. 접속 주소가 변경될 경우 별도로 안내드립니다.

| 역할 | 접속 페이지 | 비고 |
|------|-------------|------|
| 관리자 | 서비스 메인 주소 | 비밀번호 필요 |
| 고객 | 서비스 주소`/w/chat` | 로그인 불필요, 링크만으로 접속 |

> 처음 접속 시 외부 보안 안내 페이지가 나타날 수 있습니다. **"Visit Site"** 버튼을 눌러 진행해주세요.
